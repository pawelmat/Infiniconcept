Axion

About the Game
Axion is an experimental 3D tactical strategy game set against the backdrop of deep space. Work has been started in 2025 and is still in progress - it is not fully functional yet but rather shows what it can and will be.

You command a squad of combat units engaged in strategic encounters with enemy forces — where positioning, timing, and tactical decisions determine victory.

The game features a fully software-rendered 3D engine, running on both classic Amigas and modern PowerPC-based systems such as AmigaOS 4.1 and MorphOS. No hardware acceleration is used - just raw CPU power and software rendering.

Hardware Requirements (Classic Amiga)
  • Minimum requirements: CPU Motorola 68030 @ 50 MHz + OCS
  • Recommended requirements: CPU Motorola 68060 @ 100 MHz or faster + RTG
  • FPU required
  • RAM: 6 MB free
Supported Systems
  • AmigaOS 3.x (classic Amiga)
  • AmigaOS 4.1
  • MorphOS
  • Pistorm
  • Pimiga
  • WinUAE
